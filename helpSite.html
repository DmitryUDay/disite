Помощь: Хостинг сайта

Инструкция для самостоятельной публикации сайта (БЕСПЛАТНО)

Настройка вашего сайта состоит из 7 пунктов

Содержание:

1. Распаковка 2. Тестирование 3. Покупка домена 4. Выбор хостинга 5. Загрузка сайта 6. Запуск сайта 7. Подключение домена
Шаг 1

Распакуйте RAR файл

После полной оплаты работы, Kodarium пришлет вам архив - это ваш сайт но в коробке которую нужно распаковать. Откройте ее при помощи zip разархиватора и распакуйте в любое удобное место. Например на рабочий стол

Шаг 2

Проверка сайта

Убидитесь, что сайт отобржается корректно и не ломается. Для этого откройте распакованную папку и откройте файл при помощи браузера. проверьте точность информации и целостность сайт

Шаг 3

Покупка домена

Для того, чтобы ваш сайт не выглядел как (https://WebGithub/github.io/ваше_имя_сайта/123/example.com) мы сделаем короткое и красивое имя которое станет (Имя_сайта.com) Для этого советуем преобрести домен у провайдера Рег.ру После покупки домен отобразится у вас в личном кабинете

Шаг 4

Настройка или покупка хостинга

!!!Бесплатный сервис хостинга cloudflare имеет шанс блокировки РКН* ( * - мошеническая организация запрещенная на ресурсах UDay) !!! На примере возьмем бесплатный, но низкий по скорости веб хостинг GitHub pages. Для этого перейдите на сайт GitHub и зарегистрируйтесь. После вверху нажмите на (+) и выберите кнопку (создать репозиторий). Введите любое название для проекта и ниже укажите тип проекта публичный иначе сайт не будет работать.

Шаг 5

Выгрузка сайта на хостинг

После создания репозитория он откроется автоматически и предложит вам создать или опубликовать файлы. Выберите кнопку загрузка существующего файла. Перед вами появится окно, в которое мы из папки сайта должны скинуть все что находится внутри нее. После полной загрузки сайта нажмите внизу кнопку внести изменения (commit changes) и дождитесь подождите пока GitHub скачает ваш сайт

Шаг 6

Включение сайта

Когда сайт загружен нам необходимо указать гитхабуц что мы хотим сделать теперь из репозитория хостинг сайта. для этого переходим в верхнем меню в настройки и в меню выбираем Страница(pages) и в блоке (Branch) выбираем ветку main. Папку root не трогаем. Нажимаем кнопку сохранить. Гитхаб начнет создание ссылки для вашего сайта. это занимает от 3-х минут до 1 часа. После гитхаб создаст ссылку, но нам он ее не покажет. поэтому ссылка на ваш сайт будет выглядеть примерно так: https://название_вашего_аккаунта.github.io/название_вашего_репозитория/index.html.

Шаг 7

Подключение домена

Если сайт открылся значит все готово но есть нюанс. наша ссылка на сайт выглядит не красиво и сложно, и поэтому ее придется настроить. для начала идем к провайдеру домена и находим ресурсные записи. Удаляем абсолютно все и вводим добавляем с нуля по такому принципу. Добавить запись >>> А >>> subdomain(@) IP Address(вводим по порядку IP гитхаба) IP для гитхаба(нужно все вводить):

  • 185.199.108.153
  • 185.199.109.153
  • 185.199.110.153
  • 185.199.111.153
Переходим в гитхаб pages, спускаемся ниже и находим блок кастомный домен(Custom domain). Вводим туда наш домен который пердоставил провайдер например SpilDerevev.com и нажимаем Save. начнется проверка домена на пренадлежность вам. После успешной проверки Дождитесь около часа, пока гитхаб расскажэет всему интернету где ваш сайт искать. После вы можете проверить доступ к сайту по домену. Поздравляем! сайт работает бесплатно и без ограничений!
